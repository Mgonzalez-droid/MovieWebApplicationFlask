# Setup

Prerequisite: To configure this new implementation you will need to have python3 installed.

1) After installing Python3 ensure you have virtualenv installed, this will allow you to use the preconfigured virtualenv in this project repo.

'python3 -m pip install --user virtualenv'

2) You should now be able to open the virtualenv

Mac/Linux:
'source env/bin/activate'

Windows:
'.\env\Scripts\activate'

3) You should now be able to run the web app as follows:

'python3 app.py'

4) Navigate to this url:

http://127.0.0.1:5000/

issues installing virtualenv:
